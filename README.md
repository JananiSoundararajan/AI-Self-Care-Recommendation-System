
# 🌿 Self-Care AI — MVP

AI-powered self-care recommendation system. Submit a daily wellness check-in and receive a personalized self-care plan generated by a **scikit-learn classifier + ChromaDB RAG + OpenAI LLM** pipeline.

## Data Flow

```
POST /api/v1/checkin
  │
  ├─► [Pydantic] Validate & parse input
  ├─► [SQLite]   Persist check-in row
  ├─► [ML]       RandomForest classifies wellness → "low" | "medium" | "high"
  ├─► [ChromaDB] Embed user state → retrieve top-3 relevant wellness tips (RAG)
  ├─► [LLM]      LangChain prompt = mood label + 7-day history + RAG context
  │              → GPT-4o-mini generates structured JSON plan
  │              → fallback: deterministic mock plan if no API key
  ├─► [SQLite]   Persist recommendation row
  └─► [Response] Return RecommendationResponse JSON to client
```

## Quickstart — Local (No Docker)

### 1. Clone & set up Python environment

```bash
git clone <repo>
cd selfcare-ai/backend
python -m venv venv
source venv/bin/activate        # Windows: venv\Scripts\activate
pip install -r requirements.txt
```

### 2. Configure environment

```bash
cp .env.example .env
# Edit .env — add your OpenAI key, or set USE_MOCK_LLM=true to skip it
```

### 3. Train the ML model (one-time)

```bash
python -m app.ml.train
# Output: app/ml/mood_classifier.joblib
```

### 4. Run the backend

```bash
uvicorn main:app --reload --port 8000
```

The server auto-creates the SQLite DB and seeds ChromaDB on first run.

### 5. Run the frontend (separate terminal)

```bash
cd ../frontend
pip install -r requirements.txt
streamlit run app.py
```

Open **http://localhost:8501** in your browser.

## API Reference

### POST /api/v1/checkin

Submit a daily check-in. Triggers the full ML → RAG → LLM pipeline and returns a plan immediately.

**Request:**
```json
{
  "user_id": "alice",
  "mood": 4,
  "sleep_hours": 5.5,
  "stress_level": 8,
  "activity_level": 3,
  "note": "Big deadline tomorrow, feeling anxious"
}
```

**Response (201):**
```json
{
  "user_id": "alice",
  "checkin_id": 1,
  "mood_label": "low",
  "recommendations": {
    "morning": "Start with 5 minutes of box breathing before checking your phone. Eat a protein-rich breakfast to stabilize blood sugar.",
    "afternoon": "Step outside for a 10-minute walk — even brief sunlight and movement reset cortisol levels.",
    "evening": "Put screens away 30 minutes before bed. Try the 4-7-8 breathing technique to ease into sleep.",
    "focus_tip": "Your stress and sleep patterns suggest your nervous system needs recovery mode. Protect your evening aggressively."
  },
  "generated_at": "2026-04-17T08:00:00"
}
```

### GET /api/v1/recommendation/{user_id}

Fetch the user's most recent recommendation.

### GET /api/v1/history/{user_id}?limit=10

Fetch check-in history.

**Response:**
```json
[
  {
    "checkin_id": 1,
    "mood": 4,
    "sleep_hours": 5.5,
    "stress_level": 8,
    "activity_level": 3,
    "mood_label": "low",
    "created_at": "2026-04-17T08:00:00"
  }
]
```

### GET /api/v1/health

```json
{ "status": "ok", "service": "selfcare-ai" }
```

## ML Model Details

- **Algorithm:** `RandomForestClassifier` (scikit-learn)
- **Features:** mood (1-10), sleep_hours, stress_level (1-10), activity_level (1-10)
- **Labels:** `low` / `medium` / `high` (wellness state)
- **Training data:** 2,000 synthetic samples with rule-based labeling + gaussian noise
- **Pipeline:** `StandardScaler` → `RandomForestClassifier`
- **Fallback:** weighted scoring formula (no model file needed)

## LLM Integration

- **Model:** `gpt-4o-mini` (cost-efficient, fast)
- **Context injected:** mood label, 7-day history, 3 RAG-retrieved wellness tips
- **Output format:** enforced JSON via `response_format={"type": "json_object"}`
- **Fallback:** deterministic mock plans per wellness tier (no API key needed)

## ChromaDB RAG

- **Knowledge base:** 30 curated, evidence-based wellness tips
- **Embedding:** ChromaDB's default (all-MiniLM-L6-v2)
- **Retrieval:** cosine similarity, top-3 results per check-in
